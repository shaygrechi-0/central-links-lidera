# Central de Planilhas e Documentos (Academia Lidera)

## Arquivos
- `index.html` (site)
- `links.json` (lista de links)
- `assets/logo.png` (sua logo — coloque aqui)

## Como publicar no GitHub Pages
1. Crie um repositório no GitHub (ex.: `central-links`)
2. Suba **estes arquivos** na raiz do repositório
3. Vá em **Settings → Pages**
4. Em **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
5. Salve e aguarde o link do Pages aparecer.

URL padrão:
`https://SEU_USUARIO.github.io/NOME_DO_REPO/`
